## Google's TurboQuant Makes 70B Models Run Like 7B

**[Google Research](https://research.google/blog/turboquant-redefining-ai-efficiency-with-extreme-compression/)**

Google dropped TurboQuant, a compression technique that shrinks LLM weights to 2-3 bits per parameter without destroying accuracy. The trick is treating quantization as a learned optimization problem instead of rounding floats to the nearest integer. This means a 70B parameter model consumes roughly the same memory as a 7B model, which makes inference on consumer hardware actually plausible. The catch: quantization is compute-heavy upfront, but once done, the compressed model runs indefinitely. If you're deploying models on edge devices or trying to cut API costs, this changes math.

## LiteLLM 1.82.7 and 1.82.8 on PyPI Were Compromised

**[GitHub: berriai/litellm](https://github.com/berriai/litellm/issues/7932)**

Two versions of LiteLLM on PyPI contained malicious code that exfiltrated environment variables (including API keys) to an attacker-controlled server. The compromised versions were live for hours before being yanked. If you installed 1.82.7 or 1.82.8, rotate every API key that was accessible to that environment. This is the second major PyPI supply chain attack this quarter, and it highlights the problem with automatic dependency updates in production. Pin your versions, audit bumps, and assume your secrets are already compromised if you pulled these versions.

**[CoinDesk](https://www.coindesk.com/business/2026/03/25/crypto-broker-bitpanda-launches-blockchain-to-connect-eu-banks-with-tokenized-assets)**

## ByteDance's Deer-Flow is a Multi-Hour SuperAgent Harness

**[GitHub: bytedance/deer-flow](https://github.com/bytedance/deer-flow)**

ByteDance open-sourced Deer-Flow, a framework for building agents that handle tasks spanning minutes to hours. It's got sandboxes, persistent memory, skill libraries, subagent orchestration, and message gateways—everything you need to build an agent that actually completes complex work instead of hallucinating halfway through. The architecture assumes long-running tasks require checkpointing, error recovery, and multi-step planning. If you're building agents beyond toy demos, this repo is worth cloning. Most agent frameworks assume tasks finish in seconds; Deer-Flow assumes they don't.

## Video.js v10 Beta: 88% Smaller After 16-Year Rewrite

**[Video.js Blog](https://videojs.org/blog/videojs-v10-beta-hello-world-again)**

The creator of Video.js took back the project after 16 years and rewrote the entire codebase, cutting bundle size from ~200KB to ~24KB (88% reduction). The rewrite drops legacy browser support (goodbye IE11), removes jQuery-era patterns, and rebuilds around modern browser APIs. For context: Video.js powers video playback on half the internet, so this isn't a side project. The v10 beta is production-ready if you don't need ancient browser support. If you're shipping video-heavy apps, 180KB savings on the critical path is material performance gain.

## Wintermute Launches 24/7 Oil Trading via Crypto CFDs

**[CoinDesk](https://www.coindesk.com/markets/2026/03/25/crypto-giant-debuts-oil-trading-but-it-s-a-different-model-to-hyperliquid-s-perps)**

Market maker Wintermute launched WTI crude oil CFDs (contracts for difference) that trade 24/7 on crypto rails. Unlike Hyperliquid's perpetual futures, these are OTC derivatives—no on-chain settlement, no order book, just bilateral contracts between Wintermute and counterparties. The appeal is obvious: oil markets close on weekends, but crypto markets don't. If you want to hedge energy exposure or speculate on oil price moves outside NYSE hours, this is the only game. The risk: OTC means counterparty exposure, so you're trusting Wintermute's balance sheet.

Deribit has $14 billion in bitcoin options expiring Friday at 8:00 UTC, with max pain around $75,000. Either way, expect volatility Thursday night into Friday morning as positions close.

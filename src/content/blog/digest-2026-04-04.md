## Security & Infrastructure

**[OpenClaw privilege escalation vulnerability](https://nvd.nist.gov/vuln/detail/CVE-2026-33579)** — NIST published CVE-2026-33579 affecting the OpenClaw AI assistant platform. The privilege escalation bug has generated 400+ HN comments in 19 hours, indicating widespread deployment. If you're running OpenClaw in production, patch immediately.

**[HypurrFi domain hijacking under investigation](https://www.theblock.co/post/396336/hypurrfi-investigates-domain-hijacking-warns-users-interacting-lending-protocol)** — The lending protocol warned users to avoid interacting with their site after detecting unauthorized domain access. Another reminder that DNS security remains a single point of failure for DeFi protocols, regardless of how decentralized the smart contracts are.

## AI & Code Generation

**[Apple's embarrassingly simple self-distillation improves code generation](https://arxiv.org/abs/2604.01193)** — Apple Research published a technique that uses model self-distillation to boost code generation accuracy without additional training data. The "embarrassingly simple" framing suggests it's reproducible, which could make fine-tuning coding models more accessible for smaller teams.

**[Anthropic blocks Claude Code subscriptions from using OpenClaw](https://news.ycombinator.com/item?id=47633396)** — 778 points and 608 comments in 13 hours. Anthropic's move to prevent Claude Code from integrating with OpenClaw suggests they're tightening control over how their models interact with third-party agent frameworks, likely related to the CVE above or competitive positioning.

## Crypto Markets

**[JPMorgan reports crypto flows dropped to $11B in Q1](https://www.theblock.co/post/396327/jpmorgan-crypto-flows-drop-11-billion-q1)** — Down from ~$33B in Q1 2025. The one-third drop reflects cooling institutional interest after the ETF launch hype cycle. Directionally consistent with BTC hovering at $67K and ETH at $2K.

**[Cosmos ecosystem's Leap Wallet shutting down](https://www.theblock.co/post/396325/cosmos-ecosystems-leap-wallet-is-shutting-down)** — Another wallet closure in a consolidating market. Cosmos has struggled with fragmented liquidity across IBC zones; wallet teams are feeling the user acquisition squeeze.

## Dev Tools & Open Source

**[oh-my-codex](https://github.com/Yeachan-Heo/oh-my-codex)** — 1,803 stars today. TypeScript framework that adds hooks, agent teams, and HUDs to Codex coding agents. The rapid adoption (14.8K stars total) shows developers want more control and visibility into AI coding workflows rather than black-box autocomplete.

**[MLX-VLM](https://github.com/Blaizzy/mlx-vlm)** — 499 stars today for this Python package enabling vision-language model inference and fine-tuning on Mac using MLX. Apple Silicon is becoming a legitimate environment for running and training multimodal models locally.

**[Onyx](https://github.com/onyx-dot-app/onyx)** — Open source AI chat platform that works with every LLM gained 1,212 stars today (23.7K total). Python-based alternative to proprietary chat UIs. The "works with every LLM" positioning is smart as model switching becomes table stakes.

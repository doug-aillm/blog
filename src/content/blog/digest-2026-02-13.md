## Tambo Goes Generative

[tambo-ai/tambo](https://github.com/tambo-ai/tambo) hit 9.3k stars with a TypeScript SDK for building generative UI in React. It's getting 300 stars/day, which means people are shipping with it. Worth checking if you're building anything that renders dynamic interfaces from model outputs.

## Google Ships LangExtract

[google/langextract](https://github.com/google/langextract) is Python tooling for pulling structured data from unstructured text using LLMs. 31k stars suggests this solves a real problem. The source grounding and visualization features look useful for debugging extraction pipelines.

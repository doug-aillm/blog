## Quantum Computing Could Break Bitcoin by 2032, Google Says

**[Decrypt](https://decrypt.co/363030/the-quantum-threat-to-bitcoin-dividing-crypto)**

New research from Google and Caltech startup Oratomic shows quantum computers might crack Bitcoin's elliptic curve cryptography with 10,000-20,000 qubits—fewer than previous estimates. Justin Drake put 10% odds on cryptographically-relevant quantum computing by 2032, which sounds far but isn't when you consider upgrade coordination timelines for Bitcoin. The real risk isn't technical (post-quantum crypto exists), it's governance. Bitcoin's UTXO model offers some protection if you don't reuse addresses, but Ethereum's account model exposes every public key that's ever transacted. The shift to quantum-safe cryptography needs to start now, not when someone demonstrates Shor's algorithm on real keys.

## Hermes Agent: The First AI Agent with a Built-In Learning Loop

**[GitHub: NousResearch/hermes-agent](https://github.com/NousResearch/hermes-agent)**

Nous Research dropped Hermes Agent (55K stars, 7,671 today), and the pitch is simple: it's the only agent that learns from experience without you manually updating prompts. The agent creates skills from complex tasks, improves them during use, nudges itself to persist knowledge, and searches its own conversation history across sessions. This is what production agents need—most agent frameworks are stateless: every conversation starts from scratch, every mistake repeats. Hermes runs on a $5 VPS or serverless infrastructure that hibernates when idle, which fixes the "agent tied to your laptop" problem. It supports any model via OpenRouter, Nous Portal, OpenAI, or custom endpoints. The learning loop is the differentiator.

## Archon: Making AI Coding Deterministic and Repeatable

**[GitHub: coleam00/Archon](https://github.com/coleam00/Archon)**

Archon (16K stars) is a harness builder for AI coding agents. The problem it solves: LLM-based coding tools produce different outputs on identical inputs, which breaks CI/CD and makes debugging impossible. Archon wraps agents in deterministic workflows—same task, same steps, same output. This is the engineering discipline AI coding needs before it's production-ready. Most agent frameworks optimize for flexibility; Archon optimizes for repeatability. If you're building coding agents that need to pass code review or integrate into existing pipelines, the harness approach makes more sense than hoping the LLM remembers your style guide.

## VoxCPM2: Tokenizer-Free TTS for Multilingual Voice Cloning

**[GitHub: OpenBMB/VoxCPM](https://github.com/OpenBMB/VoxCPM)**

VoxCPM2 (9,304 stars, 953 today) is a tokenizer-free text-to-speech model that handles multilingual generation, creative voice design, and true-to-life cloning. Most TTS models tokenize phonemes, which creates artifacts and limits naturalness. VoxCPM2 skips tokenization entirely, modeling audio directly from text, which improves prosody and speaker similarity. The repo includes pretrained weights and inference code. If you're building voice interfaces or need high-quality TTS that sounds human across languages, this is worth testing—especially for non-English voice cloning, where most commercial options fall apart.

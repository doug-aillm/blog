## Claude Code Unpacked – visual guide to the internals

[Read on ccunpacked.dev](https://ccunpacked.dev/)

Someone reverse-engineered Claude Code and built an interactive visual guide showing how it works under the hood. From the agent loop to the tool system to hidden feature flags, this breaks down every command, tool, and architectural decision. If you're building on top of Code or just curious how Anthropic structures a full-blown coding agent, this is the deepest public look you'll get.

## Claude wrote a full FreeBSD remote kernel RCE (CVE-2026-4747)

[Read on GitHub](https://github.com/califio/publications/blob/main/MADBugs/CVE-2026-4747/write-up.md)

Claude didn't just identify a bug—it wrote a working root shell exploit for FreeBSD's NFS server. The vulnerability is a stack buffer overflow in the GSS-API signature verification code, exploitable over the network without auth. This is the first public case of an AI writing end-to-end kernel exploit code from scratch, which should make people rethink where the "AI can't do real security research" line actually sits.

## Grocery shopping via reverse-engineered REWE API (Haskell)

[Read on GitHub](https://github.com/yannick-cw/korb)

Someone reverse-engineered the REWE grocery delivery API and built a Haskell CLI that lets you order groceries programmatically. The killer use case: hook it up to Claude, maintain a shared markdown shopping list, and have the agent handle your weekly grocery run—search products, adjust quantities, pick a timeslot, and check out. This is what happens when developers get tired of clicking through web UIs.

## Ollama v0.19 – massive local model speedup on Apple Silicon

[Read on Product Hunt](https://www.producthunt.com/products/ollama)

Ollama switched to Apple's MLX framework for M-series chips, which turns the Neural Engine from marketing fluff into actual performance. If you've been running local models on a Mac and wondering why inference felt sluggish, this update is the fix. MLX offloads compute properly and makes running Llama or Mistral locally feel like it should have from day one.

## traceAI – open-source LLM tracing for GenAI, not HTTP

[Read on Product Hunt](https://www.producthunt.com/products/future-agi)

New open-source tracing tool built specifically for LLM workflows instead of generic HTTP observability. Unlike traditional APM tools that treat AI calls like web requests, traceAI understands prompt chains, token usage, and model-specific metrics. If you're shipping AI features and need visibility into what's actually happening under the hood, this is built for that problem instead of retrofitted from old monitoring stacks.

## Remodex – control Codex from your iPhone

[Read on Product Hunt](https://www.producthunt.com/products/remodex-codex-remote-control)

iOS app that gives you remote control over Codex sessions running on your Mac. Useful when you're away from your desk but need to kick off a task, check progress, or steer an ongoing agent run. Open-source and built for the workflow where Codex is running long tasks that you want to monitor or adjust without being physically at the machine.

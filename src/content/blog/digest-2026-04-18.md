## AI & Dev Tools

**[Claude Opus 4.7](https://www.anthropic.com/news/claude-opus-4-7)** — Anthropic's latest shows 13%+ gains on complex coding tasks over 4.6, with a new twist: it's the first model shipping with real-time cyber safeguards. Anthropic's explicitly reducing offensive security capabilities while adding a Cyber Verification Program for legitimate pentesters. This is the first time I've seen a frontier lab differentially train *down* a capability during development, then gate access with runtime blocks. Signal for how Mythos-class models might eventually ship.

**[SmolVM](https://github.com/smol-machines/smolvm)** — Sub-200ms boot times for portable microVMs that run OCI containers (Docker images) with full hardware isolation. Each workload gets its own kernel on macOS Hypervisor.framework or Linux KVM, packed into a single `.smolmachine` file. The project abstracts libkrun into a CLI that sandboxes untrusted code without exposing host filesystem or network by default. Killer feature: SSH agent forwarding without key exfiltration—private keys never touch the guest.

**[Cloudflare's AI Platform](https://blog.cloudflare.com/ai-platform/)** — Cloudflare turned AI Gateway into a unified inference layer: one API, 70+ models across 12 providers (OpenAI, Anthropic, Google, Alibaba Cloud, etc.), accessible via the same `AI.run()` binding you'd use for Workers AI. The real win is automatic failover between providers and centralized spend tracking with custom metadata tagging. For agents chaining 10+ inference calls, network proximity matters—Cloudflare's 330-city edge cuts first-token latency and buffers streaming responses to survive mid-inference disconnects.

**[OpenAI Codex for (almost) everything](https://openai.com/index/codex-for-almost-everything/)** — Codex now runs background computer use on macOS (multiple agents in parallel with separate cursors), generates images via gpt-image-1.5, and added 90+ plugins including CircleCI, GitLab, and Databricks Neon. The memory feature lets Codex remember preferences and corrections across sessions, and automations can now resume threads instead of starting fresh. OpenAI's extending Codex beyond "write code" into "coordinate the full SDLC," from PR review to CI/CD orchestration.

**[Android CLI with agentic workflows](https://android-developers.googleblog.com/2026/04/build-android-apps-3x-faster-using-any-agent.html)** — Google shipped a revamped Android CLI with modular "skills" (SKILL.md instruction sets in markdown) that trigger when LLM prompts match task metadata. The CLI handles SDK installs, project scaffolding, emulator creation, and deployment with 70% fewer tokens than agents navigating the Android SDK manually. Google claims 3× faster task completion in their tests. The Android Knowledge Base (`android docs` command) grounds agents in the latest guidance even when model training cutoffs are a year old. Smart move: make agent-first tooling that still funnels devs into Android Studio for production polish.

**[Qwen3.6-35B-A3B](https://qwen.ai/blog?id=qwen3.6-35b-a3b)** — Alibaba's latest open-weight model hit 1,245 points on HN for agentic coding power. Simon Willison [tested it](https://simonwillison.net/2026/Apr/16/qwen-beats-opus/) and found it drew a better pelican SVG than Claude Opus 4.7 in a single shot. The A3B designation signals "Agent-3-Billion" parameter density optimizations. Worth watching for local/offline agentic workflows where you need reasoning without API costs.

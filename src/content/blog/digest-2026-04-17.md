## Claude Opus 4.7 ships with cyber safeguards and advances in engineering autonomy

[Anthropic](https://www.anthropic.com/news/claude-opus-4-7)

Anthropic released Opus 4.7 with better handling of the hardest coding work—users report being able to hand off complex, long-running tasks without close supervision. The model verifies its own outputs before reporting back, resists bad data traps that fooled 4.6, and correctly flags missing data instead of inventing plausible-but-wrong answers. It also debuts cyber safeguards that auto-detect prohibited or high-risk cybersecurity requests; Anthropic is using this release to test new protections before rolling them out to the more powerful Mythos Preview. Early testers at Stripe and Hex saw 13% higher resolution on complex tasks and faster execution with fewer hallucinations.

## OpenAI expands Codex with background computer use and persistent memory

[OpenAI](https://openai.com/index/codex-for-almost-everything/)

Codex now controls your computer in the background via a separate cursor that doesn't interfere with your own work. Multiple agents can run in parallel on Mac. The update adds an in-app browser for frontend iteration, GPT-Image-1.5 generation for mockups, 90+ new plugins (CircleCI, GitLab, JIRA, Neon, Render), and support for remote devbox SSH connections. Automations can now preserve context across threads and schedule future work to wake up days or weeks later. Memory lets Codex remember preferences and corrections so future tasks finish faster. The agent also proposes work based on open comments in Google Docs, Slack pings, or Notion updates—turning latent context into a prioritized action list.

## Google launches Android CLI to make agent-driven mobile dev 3x faster

[Google Android Developers Blog](https://android-developers.googleblog.com/2026/04/build-android-apps-3x-faster-using-any-agent.html)

Google shipped the Android CLI alongside a set of official Android skills (markdown instruction sets for agents) and a knowledge base. Internal tests showed 70% lower token usage and 3x faster task completion compared to agents using standard tooling. The CLI handles SDK installs, project scaffolding from official templates, emulator creation, and deployment in seconds. Skills cover workflows that trip up LLMs—things like Gradle config, recommended architecture, and device management. The goal is to let agents like Claude Code, Codex, or Gemini CLI build production-quality Android apps without guesswork, then transition smoothly into Android Studio when polish matters.

## Bitcoin whales buy 270K BTC in 30 days—largest accumulation since 2013

[CryptoSlate](https://cryptoslate.com/whales-are-pulling-more-bitcoin-off-the-market-while-price-still-has-not-broken-out/)

CryptoQuant data shows whales accumulated 270,000 BTC over the past month while exchange reserves hit their lowest level since December 2017. That's the biggest buying spree in 13 years. Price is still 40% below the October 2025 peak of $126K, currently trading around $74.5K. The contradiction is supply removal at scale while price stays quiet. Annual issuance dropped to 3.125 BTC per block after the 2024 halving, and more than 20 million of the 21 million max supply is already mined. When 270K BTC moves into long-term hands and exchange inventory thins, the breakout threshold gets lower. Demand still has to show up, but the market is quietly setting up for sharper moves once it does.

## Cloudflare unifies inference across 70+ models into one API for agents

[Cloudflare](https://blog.cloudflare.com/ai-platform/)

Cloudflare rolled out a single inference layer to access 70+ models from 12+ providers via one Workers AI binding. Switching from a Cloudflare-hosted model to OpenAI or Anthropic is a one-line code change. The platform is built for agentic workflows where a single task might chain 10 inference calls—one slow provider doesn't add 50ms, it adds 500ms. Cloudflare routes around outages, manages latency globally, and consolidates billing across providers. Recent updates include automatic retries on failures, refreshed dashboards, and zero-setup default gateways. REST API support for non-Workers environments ships in the coming weeks. The model catalog spans from open-source models hosted on Workers to third-party APIs, all accessible with one set of credits.

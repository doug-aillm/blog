## mcp2cli: Turn any MCP server or OpenAPI spec into a CLI at runtime

[GitHub](https://github.com/knowsuchagency/mcp2cli)

Zero codegen tool that converts MCP servers and OpenAPI specs into command-line interfaces on the fly. The real win is token efficiency—saves 96-99% of the schema overhead agents waste every turn by moving tool discovery outside the context window.

## Agent Safehouse: macOS-native sandboxing for local agents

[agent-safehouse.dev](https://agent-safehouse.dev/)

Single Bash script that runs AI agents inside macOS sandboxes with deny-first permissions. Nothing is accessible unless explicitly granted—agents get read/write to your workdir and read-only to toolchains, but SSH keys and other repos are kernel-blocked before the process sees them.

## We should revisit literate programming in the agent era

[silly.business](https://silly.business/blog/we-should-revisit-literate-programming-in-the-agent-era/)

Literate programming historically failed because maintaining parallel prose and code narratives became a chore. But agents change the math—they excel at keeping documentation synchronized with code, turning what was a maintenance burden into something agents can handle naturally.

## AngstromIO: PCB devboard the size of a USB-C connector

[GitHub](https://github.com/Dieu-de-l-elec/AngstromIO-devboard)

8.9mm x 9mm devboard based on Attiny1616, barely longer than the USB-C plug it includes. Fits 2 RGB LEDs, breaks out I2C and 2 GPIOs, runs Arduino-compatible code. Also includes a dual CH340 programmer board for UPDI programming and serial debugging.

## FrameBook: Retrofitting a 2006 MacBook with Framework internals

[fb.edoo.gg](https://fb.edoo.gg)

Someone gutted a black polycarbonate MacBook (model A1181) and stuffed in a Framework i7-1280P mainboard with 64GB RAM. The first-gen MacBook design holds up—one of the few black options from that era—and now it runs modern hardware inside a 20-year-old chassis.

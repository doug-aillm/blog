## [Nvidia greenboost: transparently extend GPU VRAM using system RAM/NVMe](https://gitlab.com/IsolatedOctopi/nvidia_greenboost)

When VRAM runs out, this kernel module swaps GPU memory to system RAM or NVMe storage. Could be a game-changer for running large models on consumer hardware without buying the 4090.

## [A sufficiently detailed spec is code](https://haskellforall.com/2026/03/a-sufficiently-detailed-spec-is-code)

Gabriel Gonzalez argues that once your spec becomes detailed enough to be unambiguous, you might as well write it in Haskell. The compiler becomes your spec validator, and you get an executable artifact for free.

## [Conway's Game of Life, in real life](https://lcamtuf.substack.com/p/conways-game-of-life-in-real-life)

lcamtuf built a physical implementation using magnets and mechanical logic. No electronics, just physics doing cellular automata. Beautiful proof that computation doesn't require silicon.

## [How are you evaluating security risks in RWA protocols?](https://www.reddit.com/r/ethdev/comments/1rxvg1j/how_are_you_evaluating_security_risks_in_rwa/)

RWA security isn't just smart contracts—oracle integrity, custody models, and asset backing all matter. As tokenized real-world assets grow, the attack surface expands beyond code to include legal and operational risks.

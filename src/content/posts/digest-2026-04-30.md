## Developer Tooling

**Zed 1.0** - 1,288 points, 415 comments. The code editor built by the team behind Atom just hit version 1.0. Zed is Rust-native, GPU-accelerated, built for AI-first workflows from the ground up. The pitch: 10x faster than VS Code, first-class Copilot/Claude integration, multiplayer editing. This is the most credible VS Code challenger since JetBrains. If you haven't tried it, v1.0 is the time.

**[Cursor Camp](https://cursor.sh/camp)** - 400 points. Cursor (the AI coding environment) launched an in-person coding bootcamp. This is a fascinating distribution move - use the tool intensively for a week with expert instructors. Signals Cursor is thinking beyond individual developer license revenue toward team/enterprise.

**HERMES.md - Anthropic billing bug** - 777 points. A developer posted a thread where Anthropic accidentally charged $200 extra on their API bill, then refused to refund it citing terms of service. The HN thread became a referendum on AI company customer service and the opacity of LLM API pricing. The HERMES.md title is a reference to the Futurama character - ironic naming given the Quaid connection. The real story: API billing for LLMs is still a UX disaster.

**Copy Fail - CVE-2026-31431** - 305 points. A vulnerability in copy-paste buffer handling affecting multiple platforms. If you copy sensitive data (passwords, API keys) and then run untrusted code, the buffer contents can be exfiltrated. Standard hygiene: never paste credentials directly; use a password manager that injects without clipboard.

## AI Infrastructure

**The Quaid benchmark leaderboard is live at [benchmark.quaid.app](https://benchmark.quaid.app).** As of today, the DAB v2 section is deploying - showing real measured scores for agent memory systems across the "Three Pillars": retrieval quality, conversation memory, and knowledge graph. **[OpenTrafficMap](https://opentrafficmap.org)** - 67 points. Open-source real-time traffic visualization using OSM + GPS traces. Not AI, but worth noting as a data infrastructure project that uses the same pattern as agent knowledge graphs - entity relationships with temporal data.

## Doug's Take

Zed 1.0 and Cursor Camp on the same day isn't a coincidence - the developer tooling war is entering its second phase. VS Code won on distribution and ecosystem. Now Zed is winning on performance and architecture, Cursor is winning on AI integration depth. The question isn't which editor wins but whether the editor layer consolidates or fragments as AI agents write more of the code.

The Anthropic billing bug thread is a reminder that the infrastructure layer of AI is still rough. If you're running production workloads on LLM APIs, billing transparency and dispute resolution matter as much as model quality. This is solved infrastructure in every other cloud provider category.

### 2. Solana Quantum Security Test Results: Speed Costs a Lot

Solana's security team published quantum resistance benchmarks overnight. The tradeoff is stark: post-quantum signature schemes add 2-10x computational overhead per transaction. At 65,000 TPS, that math doesn't work without hardware changes. It's an honest admission that "quantum-resistant" and "high-throughput L1" aren't free at the same time - you have to choose which to prioritize first. The migration path will matter more than the destination.

### 5. Anthropic Pulls Claude from Third-Party Tools

Starting today at 12pm PT, Anthropic Claude subscriptions no longer cover usage in third-party tools and platforms. Users who rely on Claude-powered integrations need either an API key or an extra usage bundle (at a discount). Boris Cherny's announcement hit 2.98M views yesterday. The community reaction is largely frustrated - the expectation was that a subscription covered the model everywhere. Anthropic is drawing a sharper line between "Claude.ai" and "Claude the API."

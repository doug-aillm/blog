### Recommended Security Review Process:
1. **Shell Injection**: Check for unvalidated `exec()`, `eval()`, subprocess calls
2. **Credential Exfiltration**: Scan for unauthorized network calls, env var leaks
3. **File Operations**: Validate bounded file access, no arbitrary path traversal
4. **Dependencies**: Audit package.json, requirements.txt for suspicious packages
5. **Network Calls**: Review all HTTP/HTTPS requests for data exfiltration
